# 本地部署指导

***推荐使用pycharm打开本项目文件夹，下列命令在pycharm终端输入***

1. ```cd frontend```
2. ```npm install```
3. ```npm run build```
4. ```cd ..```
5. ```python manage.py runserver 8000```
